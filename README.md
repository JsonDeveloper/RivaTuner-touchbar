# Requirements: 
- Memcached server (on Mac) - [How to install](https://www.hacksparrow.com/install-memcached-on-mac-os-x.html)
- BetterTouchTool (on Mac) - [Official site](https://folivora.ai)
- MSI Afterburner with Rivatuner Statistics Server (on Windows PC) - [Official site](https://www.msi.com//page/afterburner)

# Rivatuner Statistics Server configuration
Set you settings in app.js file
```javascript
const config = {
  api: {
    url: "http://{you windows pc local IP adress}:82/mahm",
    username: "MSIAfterburner",
    password: "{Riva tuner secret key}"
  },
  [...]
};
```
